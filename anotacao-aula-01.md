# Operadores

## Comparação

```
IGUAL       (1 == 1) verdade
            (1 == 2) falso
NÃO IGUAL   (1 != 2) verdade 

MAIOR       (2 > 1)  verdade
            (2 < 1)  false
MENOR       (1 < 2)  verdade     

MAIOR IGUAL (2 >= 2)
MENOR IGUAL (1 <= 1)
```
